## Installation
- npm / npx / yarn

## Initiate

Initiate project:
```
npx create-react-app .
```

## Add Dependencies
Install Restful Client: 
```
npm add axios
```

## Install All Dependencies
Download all dependencies:
```
yarn install
```

## Start the services.
```
yarn start
```

## Delete unecessary (for this project)
- App.test.js
- index.css
- logo.svg
- setupTests.js

```
import './index.css';
```

```

```
